# Stage Cafe Management System

## Description

A cloud-based Cafe Management System built using Angular, Node.js, Express, Prisma, and PostgreSQL.

## Features

- Authentication
- Role-Based Access Control
- Dashboard Analytics
- Product Management
- Inventory Management
- Point of Sale (POS)
- Customer Management
- Reservations
- Payments
- Reports
- Audit Logs

## Technology Stack

Frontend
- Angular
- Angular Material
- Tailwind CSS
- TypeScript

Backend
- Node.js
- Express
- Prisma

Database
- PostgreSQL (Neon)

Deployment
- Frontend: Vercel
- Backend: Render
- Database: Neon
