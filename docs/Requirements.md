# Stage Cafe Management System

## Project Overview

The Stage Cafe Management System is a web-based application that enables Stage Cafe to manage daily business operations, including product management, inventory, orders, payments, reservations, customer records, and reporting.

---

# Objectives

- Digitize cafe operations
- Improve order processing
- Manage inventory automatically
- Track sales
- Generate reports
- Manage employees
- Improve customer experience

---

# User Roles

## Administrator

Permissions

- Manage Users
- Manage Roles
- Manage Products
- Manage Categories
- Manage Inventory
- Manage Suppliers
- View Reports
- Configure System Settings

---

## Manager

Permissions

- View Dashboard
- Manage Products
- Manage Inventory
- View Reports
- View Orders

---

## Cashier

Permissions

- Create Orders
- Receive Payments
- Print Receipts
- View Products

---

## Kitchen Staff

Permissions

- View Kitchen Orders
- Update Order Status
